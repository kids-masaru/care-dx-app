## はじめに

### アプリの概要

「介護DX カカナイ」は、介護業務のドキュメント作成を自動化するWebアプリケーションです。

**3つの主要機能：**

| 機能 | 説明 |
|------|------|
| 🎙️ 運営会議録 | 音声データから議事録を自動作成 |
| 🎙️ サービス担当者会議議事録 | 音声データから議事録を自動作成 |
| 📄 アセスメントシート | PDFから情報を抽出しスプレッドシートに転記 |

### 必要な準備

- **音声データ**：スマートフォンやICレコーダーで録音したファイル（MP3, M4A, WAV形式）
- **PDF書類**：介護保険証、主治医意見書などのスキャンデータ
- **Googleアカウント**：スプレッドシートへの書き込みに使用

## 運営会議録の作成

### 手順

1. **シートタイプを選択**
   - サイドバーで「運営会議録」を選択

2. **会議情報を入力**
   - 開催日：カレンダーから選択
   - 開催場所：会議室など
   - 開催時間：開始〜終了時間を選択
   - 参加者：出席者名をカンマ区切りで入力

3. **音声ファイルをアップロード**
   - 「音声ファイルを選択」エリアにファイルをドラッグ＆ドロップ
   - または「Browse files」ボタンでファイルを選択

4. **AI抽出を開始**
   - 「🚀 AI抽出開始」ボタンをクリック
   - 処理完了まで待機（数分かかる場合があります）

5. **結果を確認**
   - 抽出された議事録が表示されます
   - 自動的にGoogleスプレッドシートに転記されます

### 自動保存機能

- アップロードした音声ファイルは、指定のGoogle Driveフォルダに自動保存されます
- ファイル名：`YYYYMMDD_HHMMSS_元のファイル名`

## サービス担当者会議議事録の作成

### 手順

1. **シートタイプを選択**
   - サイドバーで「サービス担当者会議議事録」を選択

2. **会議情報を入力**
   - 担当者名：ケアマネージャー名
   - 利用者名：対象者の氏名
   - 開催日：カレンダーから選択
   - 開催場所：自宅、事業所など
   - 開催時間：開始〜終了時間を選択
   - 開催回数：第1回、第2回など

3. **音声ファイルをアップロード**
   - 会議を録音した音声ファイルを選択

4. **AI抽出を開始**
   - 「🚀 AI抽出開始」ボタンをクリック

5. **結果を確認**
   - 以下の項目が自動抽出されます：
     - 本人・家族の意向
     - 心身・生活状況
     - 各事業所の役割分担
     - 福祉用具・住宅改修の検討内容
     - 会議の結論

### 出力フォーマット

議事録は以下の形式で作成されます：

```
①【本人及び家族の意向】
・本人⇒「...」
・家族⇒「...」

②【心身・生活状況】
・身体状況⇒...
・精神状況⇒...
・生活状況⇒...

③【会議の結論・ケアプラン詳細】
...

④【各事業所の役割分担と確認事項】
＊事業所名A⇒...
＊事業所名B⇒...

⑤【福祉用具・住宅改修等に関する検討事項】
...
```

## アセスメントシートの作成

### 手順

1. **シートタイプを選択**
   - サイドバーで「アセスメントシート」を選択

2. **入力モードを選択**
   - 「PDFから転記」を選択

3. **PDFファイルをアップロード**
   - 介護保険証、主治医意見書などのPDFをアップロード
   - 複数ファイルを同時にアップロード可能

4. **スプレッドシートIDを入力**
   - 転記先のGoogleスプレッドシートのIDを入力
   - URLの `/d/` と `/edit` の間の文字列がIDです

5. **AI抽出を開始**
   - 「🚀 AI抽出開始」ボタンをクリック

6. **結果を確認**
   - 抽出されたデータが表示されます
   - 自動的にスプレッドシートに転記されます

### 抽出される情報

- 利用者基本情報（氏名、生年月日、住所など）
- 介護保険情報（被保険者番号、要介護度など）
- 主治医情報
- 既往歴・現病歴
- ADL・IADL情報
- その他アセスメント項目

## よくある質問（FAQ）

### Q: 音声ファイルの長さに制限はありますか？
A: 長時間の音声でも処理可能ですが、1時間を超える場合は処理に時間がかかります。

### Q: 対応している音声形式は？
A: MP3, M4A, WAV形式に対応しています。

### Q: PDFの文字が読み取れない場合は？
A: スキャン品質が低いとOCR精度が下がります。できるだけ高解像度でスキャンしてください。

### Q: 抽出結果を修正できますか？
A: 転記先のGoogleスプレッドシートで直接編集できます。

### Q: エラーが発生した場合は？
A: ページをリロードして再度お試しください。それでも解決しない場合は管理者にお問い合わせください。
